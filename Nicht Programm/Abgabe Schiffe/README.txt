# Schiffe_Versenken
***
## Allgemeine Information
Schiffe_Versenken ist ein Spiel bei dem jeder Spieler eine 
Flotte von Schiffen vor seinem Gegner versteckt mit dem Ziel, zuerst die Schiffe 
des Gegners zu versenken.

## Ausführung
Um das Spiel zu staten muss Java 8 installiert sein.
Außerdem müssen Assertion aktiviert werden.
Das erreicht man, indem das Spiel über die Konsole aufgerufen wird.
Dazu geht man über die Konsole in den Ordner, indem das Spiel ist 
und ruft es auf.

Genauer gesagt ruft man die Konsole auf, wechselt mit cd und den Pfad des Ordners
in den Ordner mit der Spiel und ruft dieses auf mit: java -ea -jar Schiffe_Versenken.jar
